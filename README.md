# Oxford Palliative Medicine — AI Exam & Flashcards

AI-powered exam questions and spaced repetition flashcards from the **Oxford Textbook of Palliative Medicine, 6th Edition** (Cherny, Fallon, Kaasa, Portenoy, Currow — Oxford University Press, 2021).

Live at: **[henryhe.me/oxfordpal](https://henryhe.me/oxfordpal)**

---

## Features

- **87 chapters** covering the full textbook — pain, GI, respiratory, psychiatric, ethics, terminal phase, non-cancer illnesses, and more
- **AI question generation** via the Anthropic API (Claude Sonnet) — questions sourced exclusively from textbook text
- **Offline flashcard deck** — every generated question is saved to localStorage automatically
- **Spaced repetition (SM-2)** — Anki-compatible algorithm schedules reviews based on performance
- **Due card study mode** — work through your due cards with Again / Hard / Good / Easy ratings
- **Session history & persistent stats** — accuracy by chapter, weakest topics, streak tracking
- **API usage tracker** — estimates remaining calls in the current 5-hour reset window
- **Light / dark mode** — auto-detects system preference, toggle in header
- **Export** — download your full card deck as JSON for backup

## Deployment (GitHub Pages)

### Option A — Subdirectory under henryhe.me (recommended)

1. Create a new repo: `github.com/henryhe1/oxfordpal`
2. Push this folder to `main`
3. In repo **Settings → Pages**: set Source to **GitHub Actions**
4. In your main `henryhe1.github.io` repo, add a CNAME or path config so `henryhe.me/oxfordpal` serves this repo

   The simplest approach with GitHub Pages + custom domain:
   - This repo deploys to `henryhe1.github.io/oxfordpal` automatically
   - Since your custom domain `henryhe.me` is already pointed at `henryhe1.github.io`, the path `/oxfordpal` will work automatically once deployed

5. Push and the GitHub Actions workflow handles the rest.

### Option B — Standalone subdomain

Deploy to `oxfordpal.henryhe.me` by adding a CNAME in your DNS:
```
oxfordpal CNAME henryhe1.github.io
```
Then add a `CNAME` file to this repo containing `oxfordpal.henryhe.me`.

## API usage notes

Questions are generated by calling the Anthropic API from your browser. The app tracks calls locally and estimates remaining capacity based on Claude's 5-hour rolling reset window:

| Plan | Estimated calls/window |
|------|----------------------|
| Free | ~20–25 (conservative estimate) |
| Pro ($20/mo) | ~45+ |
| Max ($100/mo) | Much higher |

The counter resets automatically when the 5-hour window expires. Since Anthropic does not expose a usage API, this is an estimate only.

---

## Data & Privacy

All data (saved cards, SRS state, stats, settings) lives in your browser's `localStorage`. Nothing is sent to any server except the Anthropic API when generating questions. The textbook content embedded in `sections-data.js` is used only as context for question generation.
